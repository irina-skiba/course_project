# FAQ
Самые частые проблемы при оформлении пояснительной записки курсового проектирования.
## 1. Код
1. Шрифт _**Courier New**_.
2. **По тексту** (и заголовки) шрифт - **14пт**.

### 1.1 Блок кода
1. Шрифт **12пт**.
2. Выделяется **пробельными строками**.
3. **Начало** с **абзацной** строки.
4. Выравниванивание **по ширине**.

## 2. Вложенные списки
*В СТП стр. 159 [СТП](https://www.bsuir.by/m/12_100229_1_185678.pdf) 
и в примере оформления по этому же СТП на сайте БГУИР [Стандарт предприятия (дипломные работы)  СТП 01-2024 - формы документов для студентов](https://www.bsuir.by/ru/uchebni-otdel/normativnye-dokumenty) разное оформление,
